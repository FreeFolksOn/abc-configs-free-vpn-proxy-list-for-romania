## Ce este Abc Configs?

Acest depozit GitHub conține un singur fișier — acest `README.md` — care este rescris de botul nostru **la fiecare 10 minute** cu cele mai noi 30 configurații. ## Cum folosești o configurație

1. **Copiază** orice celulă `Configurație` din tabelul de mai sus (șirul lung `vless://…` / `vmess://…` / `ss://…`).
2. **Lipește-o** într-un client compatibil cu V2Ray:
   - **Android:** v2rayNG, NekoBox, Hiddify Next, Husi.
   - **iOS:** Streisand, Shadowrocket, Hiddify, FoXray.
   - **Windows:** v2rayN, NekoRay, Hiddify, Clash Verge.
   - **macOS:** V2Box, NekoRay, ClashX Meta, Hiddify.
   - **Linux:** NekoRay, Hiddify, Clash Meta.
3. **Conectează-te.** Dacă un nod este lent sau nefuncțional, încearcă-l pe următorul — configurații proaspete apar la fiecare câteva minute.

## Întrebări frecvente (FAQ)

### Aceste configurații VPN / proxy sunt cu adevărat gratuite?

Da. Sunt colectate din canale publice de Telegram și partajate aici gratuit, fără înregistrare și fără reclame. Conținutul acestui README este pus în domeniul public sub [licența The Unlicense](https://unlicense.org/).

### Cât de des se actualizează lista?

Acest `README.md` este rescris **la fiecare 10 minute** cu cele mai recente 30 configurații. Pentru actualizări în timp real, urmărește [@abc_configs](https://t.me/abc_configs) pe Telegram, unde configurațiile noi sunt publicate în câteva secunde după colectare.

### Ce protocoale sunt acceptate?

V2Ray (VMess), VLESS (inclusiv Reality), Trojan, Shadowsocks (SS) și MTProto. Orice URI standard care începe cu `vmess://`, `vless://`, `trojan://`, `ss://` sau un link Telegram MTProto este acceptat de clienții principali.

### O configurație nu funcționează, de ce?

Proxy-urile gratuite au o durată de viață scurtă și imprevizibilă. Un nod care funcționează acum poate fi supraîncărcat sau blocat în minutul următor. Încearcă pur și simplu alt rând din tabel sau ia unul mai proaspăt de la [@abc_configs](https://t.me/abc_configs). Lista se rotește constant.

### Este sigur să folosesc aceste proxy-uri?

Acestea sunt servere terțe operate de persoane necunoscute, așa că tratează-le ca pe orice VPN gratuit: **nu** te autentifica în conturi sensibile, **nu** trimite trafic necriptat și preferă configurațiile cu TLS (VLESS+TLS / Trojan / Reality). Folosește-le pentru a ocoli cenzura, nu pentru date sensibile.
